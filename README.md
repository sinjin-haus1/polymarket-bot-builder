# Polymarket Bot Builder

No-code platform for building and deploying Polymarket trading bots.

## Description

A faceless SaaS that lets users create automated trading bots for Polymarket without writing code. Select indicators, set thresholds, and deploy - that's it.

## Features

- **Indicator Library**: Pre-built TradingView-style indicators (RSI, MACD, Moving Averages, Bollinger Bands)
- **Visual Strategy Builder**: Drag-and-drop condition builder for creating trading strategies
- **Paper Trading Mode**: Test strategies without risking real money
- **One-Click Deployment**: Deploy to cloud and run 24/7
- **Performance Analytics**: Track win rate, profit/loss over time
- **Market Monitoring**: Watch multiple markets simultaneously

## Tech Stack

- NestJS (Backend API)
- MongoDB (Database)
- GraphQL (API)
- Next.js 14 (Frontend - upcoming)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run start:dev
```
