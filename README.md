# Polymarket Bot Builder

No-code platform for building and deploying Polymarket trading bots.

## Features

- **Indicator Library** - Pre-built TradingView-style indicators (RSI, MACD, Moving Averages)
- **Visual Strategy Builder** - Drag-and-drop condition builder
- **Paper Trading Mode** - Test strategies without real money
- **One-Click Deployment** - Deploy to cloud and run 24/7
- **Performance Analytics** - Track win rate, profit/loss over time
- **Market Monitoring** - Watch multiple markets simultaneously

## Tech Stack

- NestJS (Backend)
- GraphQL API
- MongoDB
- Next.js 14 (Frontend)

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run start:dev
```
